# MyFinancialPlan Selenium project

## About

MyFinancePlan is a SaaS website for creating and tracking personal financial plans. It allows users to build and manage their own plans by tracking income and expenses, view other users’ plans (if sharing access is enabled), access median financial data based on their status for comparison, and receive actionable steps alerts to help implement and monitor plans according to their path to goals and dreams.
